# Music Player Application

This `music player application` is designed to manage your music playlists. Users can add, remove, save songs to playlists with user-defined names. This application was built using Java for the backend and JSwing for the UI. Happy listening!

## App Walk-Through

Clone the repo.

`git clone https://github.com/paige-ingram/MusicPlayerApplication.git`

Navigate and run the `main` file. 

`src-->main-->UI-->Main`

The music player application will launch in a new window.

<img width="1175" alt="image" src="https://github.com/paige-ingram/MusicPlayerApplication/assets/77078416/765d0398-be31-41f1-a0d4-5ed4b936c584">

Select 'New Playlist' to create a new playlist. The new playlist will show in the top display.

<img width="1175" alt="image" src="https://github.com/paige-ingram/MusicPlayerApplication/assets/77078416/c48da2a1-e372-44d4-9355-1356bfff7f19">

Select 'Add Song' to add songs to your playlist. Specify the name and artist of the song you are adding. A confirmation will show.

<img width="554" alt="image" src="https://github.com/paige-ingram/MusicPlayerApplication/assets/77078416/d643e366-dbbf-4022-9315-3ea5f7d4e7a5">

Select 'View Playlist' to view all the songs in one of your playlists. 

<img width="350" alt="image" src="https://github.com/paige-ingram/MusicPlayerApplication/assets/77078416/ec81d6c2-78d8-4a96-a6f7-c4f89cc34426">
<img width="367" alt="image" src="https://github.com/paige-ingram/MusicPlayerApplication/assets/77078416/62ad5eda-bd0e-47fb-8399-44c1955bd693">
<img width="353" alt="image" src="https://github.com/paige-ingram/MusicPlayerApplication/assets/77078416/63a4ca7d-b7ae-4ca2-b57e-5415d0ea6c45">
<img width="312" alt="image" src="https://github.com/paige-ingram/MusicPlayerApplication/assets/77078416/b1c9f363-7cf3-4661-9da1-226977e604ad">

Select 'Save Music Player' for data persistence after the application stops running. Select 'Load Music Player' to load your saved playlists when you re-run the application. The data are stored in JSON files in the data directory.
